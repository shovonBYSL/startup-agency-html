overview

Awardy Go- Creative Agency HTML Template

The Creative Agency HTML Template is an excellent option for creative agencies looking to showcase their work and services in a modern and engaging way. Built using HTML5, the template is highly customizable and easy to use, with a flexible layout system that allows users to create unique and engaging page designs. Key features include a responsive and mobile-friendly design, multiple pages, smooth animations and transitions, and built-in support for popular third-party libraries and tools. With detailed documentation and code comments included, users of all experience levels can quickly and easily modify the template to suit their needs and create a stunning and effective online presence for their agency.

Change Basename and Route

You can change the route. Go to project folder and make your route as your need.

Change Site Title and Favicon

Go to every HTML file (ex: index.html) and change the title inside header tag. Go to /src/assets/images folder to change the fav icon

Change Logo

Go to every Html file (ex: index.html) and change logo.

Customize Menu

Go to every Html file (ex: index.html) and change menu items.

Customize Footer

Go to every Html file (ex: index.html) and change footer logo, social media, footer data.

Change Copyright Infomartaion

Go to every Html file (ex: index.html) and change copyright information.

Typography - Font Family

Go to /src/assets/styles/global.css file and put your font family link to change the font of the website.

Typography - Color

Go to /tailwind.config.js file and put your color code to change font color and other colors.
